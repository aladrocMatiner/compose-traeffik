## ADDED Requirements

### Requirement: Wiki.js docs maintain multilingual parity
The system SHALL provide Wiki.js service documentation and root README references in EN/SV/ES with consistent anchors and language selectors.

#### Scenario: User switches Wiki.js docs language
- **WHEN** a user opens the Wiki.js service page in EN, SV, or ES
- **THEN** each page contains the standard service anchors and links to the equivalent language variants
- **AND** root README service links point to the matching language file for Wiki.js
