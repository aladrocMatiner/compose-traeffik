# Scripts (Gentoo QEMU Experimental)

Put qualification scripts/spikes here first. Suggested scripts:
- `fetch-image.sh` (download + checksum verify)
- `inspect-image.sh` (basic metadata and quick checks)
- `smoke-boot-libvirt.sh` (boot with cloud-init and capture evidence)
- `collect-evidence.sh` (copy logs/outputs into `artifacts/runs/<timestamp>/`)
